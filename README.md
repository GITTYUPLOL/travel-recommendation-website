# TravelMate — Travel Recommendation Website

A small, fast travel-recommendation site built with vanilla HTML, CSS, and
JavaScript. Search a curated catalog of beaches, temples, and countries.

Built for the JavaScript Programming Essentials final project.

## Pages

- **Home** (`index.html`) — landing page with hero, search bar, and the
  destinations grid.
- **About Us** (`about.html`) — company info and usage guidance.
- **Contact Us** (`contact.html`) — contact info and a working email form.

## Search

Type a keyword (e.g. `beach`, `temple`, `country`) into the home-page search
bar and click **Search**. Click **Clear** to reset.

## Run locally

It's a static site — open `index.html` in any browser, or serve the directory:

```
python3 -m http.server 8000
```
